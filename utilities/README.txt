---------
utilities
---------

boundary_filter_optimizer.m: script used the design the optimal filter window function for wavesim, minimizing the reflections at the boundary layer.

efficient_wiggle_sequence: script used to find an efficient set of ACC sequences for an n-dimensional open simulation.

homogeneous_medium_analytic_solution.m: compute the analytic solution to the Helmholtz equation for a 1D homogeneous medium
